Chronicle
desktop companion · app windows #1–#8
A tiny macOS app that sits beside any project Claude Code is building from a written plan. It answers four things at a glance for a non-developer: where the build is, what needs the operator, what to paste and where, and lets them act — file explorer + real terminals — in one window. Every window is manifest-driven by a chronicle.json a Claude session writes. Dark-canonical v1.
done here / needs you error later colour only ever signals state · grey is everything else
Real Weave (shadcn · new-york) mounts: Button · Tabs · Command · Alert. The signature compositions [★] — the loom rail, status word, file chip, paste row, terminal strip, tree row, resize handle — are composed on the same tokens and annotated per region. Drag the hairline between any two panes to set column widths.
