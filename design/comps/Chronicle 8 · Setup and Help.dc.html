Setup & Help — G1–G8
Two app-level surfaces for a non-developer: a Setup screen that installs the five things Chronicle needs — with no typed commands — and a Help screen of task recipes and a plain glossary. Every word is read as the non-developer will read it: no PATH / CLI / npm / git in a headline. Tokens by name; the only chroma is the state palette and the two agent marks. Comps match the shipped app — where a frame would diverge from Decks 1–7, the app's convention wins and the note says so.
